# OpenForge

OpenForge is an open-source ecosystem of modular developer services.

Instead of building every tool from scratch, developers can create, contribute to, and deploy independent services that solve specific problems—from automated repository documentation and embeddable AI chatbots to future workflow automation, analytics, monitoring, and developer productivity tools.

Each service is designed to be:

* **Independent** – Built and maintained as its own project.
* **Open Source** – Community-driven and extensible.
* **Self-Hostable** – Run locally or deploy to your preferred infrastructure.
* **Developer-Focused** – Designed to simplify development workflows and operations.

## Current Services

### Onboarder

AI-powered repository documentation that automatically tracks code changes, maintains file-level narratives, and keeps project documentation up to date through automated pull requests.

### Chatbot Service

A hosted, embeddable chatbot platform that allows developers to create AI assistants for static websites using a single script tag and their own knowledge base.

## Vision

OpenForge aims to become a collection of reusable developer services that can be independently developed, shared, and deployed while working together as part of a broader ecosystem.

Build once. Share openly. Forge together.
